What is the <title> tag?
<title> is an HTML tag used to set the name of the webpage.
This name appears on:
Browser tab
Browser history
Browser bookmarks
Search engine results (Google title)
Where do we use it?
The <title> tag is written inside the <head> section of an HTML document.
Syntax
<!DOCTYPE html>
<html>
<head>
    <title>This is my webpage</title>
</head>
<body>
    <h1>Welcome</h1>
</body>
</html>

Why is the <title> tag important?
✔ 1. Identifies the page
Users see the title on the browser tab.
✔ 2. SEO (Search Engine Optimization)
Google uses the title to show your webpage in search results.
✔ 3. Bookmark name
When someone saves your page, the title becomes the bookmark name.
✔ 4. Accessibility
Screen readers read the title first for blind users.
